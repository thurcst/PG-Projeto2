# Projeto Ray-Tracing

## Etapas

Este projeto foi dividido em 3 Notebooks por cada versão precisar de diferentes leituras e features.

- Cada versão se encontra no arquivo RT@.ipynb, sendo eles do 1 ao 3.

## Especificações

Cada Notebook contém as específicações da versão e todas as funções estão comentadas e de fácil leitura.

##### As features bônus não foram desenvolvidas.

## Observações

- Os casos de teste não foram todos plotados por questão de tempo computacional, alguns arquivos levavam horas para ser computado e não havia máquina disponível para aguardar até o final.
